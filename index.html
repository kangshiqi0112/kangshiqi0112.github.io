Matplotlib：专门用于开发2D图表（包括3D图表）库

  1. 为什么使用？
    • 数据可视化
    • js库，D3，echarts
  2. 三层结构
    • 容器层：1.画板层（Canvas） 2.画布层（Figure）：plt.figure() 3.绘图区/坐标系（axes）：plt.subplots()
    • 辅助显示层：网格、刻度、图例、标题等辅助显示，使图表清晰易懂
    • 图像层：画各种类型、风格图表
  3. 折线图绘制
    • 导包，包含了一系列绘图函数import matplotlib.pyplot as plt
    • 图表保存 plt.savefig(“test1.png”)
    • 绘制画布 plt.figure()
    • 绘制折线 plt.plot(x,y)
    • 显示图例 plt.legend()
    • 设置坐标轴标签，图表标题 plt.xlabel() plt.ylabel() plt.title()
    • 修改坐标轴刻度信息 plt.xticks() plt.yticks(y,**args)
    • 添加网格显示 plt.grid()
    • 图表显示 plt.show()
import matplotlib.pyplot as plt  #导包，包含了一系列绘图函数
%matplotlib inline  #环境，有些不需要
import random
x = range(60)
y_shanghai = [random.uniform(15,18) for i in x]
y_nanchang = [random.uniform(10,16) for i in x]
plt.figure(figsize=(20,8),dpi=80) #设置大小和清晰度
plt.plot(x,y_shanghai,color='r',linestyle='-.',label='shanghai')
plt.plot(x,y_nanchang,color='b',linestyle='--',label='nanchang')
#显示图例
plt.legend(loc='lower left') 
#修改坐标刻度值
x_lable =["11点{}分".format(i) for i in x] #x轴说明
plt.xlabel("time") #x轴标签
plt.ylabel("temp") #y轴标签
plt.title("picture") #图表标题
plt.yticks(range(0,40,5))#y轴0-40，隔5
plt.xticks(x[::5],x_lable)
#添加网格显示
plt.grid(True,linestyle='--',alpha=0.5)#风格和透明度，true可省略
plt.show()#之后会释放所有图像资源

newdraft

Markdown是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容具有一定的格式。

相比WYSIWYG编辑器

优点：
1、因为是纯文本，所以只要支持Markdown的地方都能获得一样的编辑效果，可以让作者摆脱排版的困扰，专心写作。
2、操作简单。比如:WYSIWYG编辑时标记个标题，先选中内容，再点击导航栏的标题按钮，选择几级标题。要三个步骤。而Markdown只需要在标题内容前加#即可

缺点：
1、需要记一些语法（当然，是很简单。五分钟学会）。
2、有些平台不支持Markdown编辑模式。

还好，简书是支持Markdown编辑模式的。

开启方式：设置->默认编辑器->Markdown编辑器

在想要设置为标题的文字前面加#来表示
一个#是一级标题，二个#是二级标题，以此类推。支持六级标题。

注：标准语法一般在#后跟个空格再写文字，貌似简书不加空格也行。
