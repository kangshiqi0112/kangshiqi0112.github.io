Matplotlib：专门用于开发2D图表（包括3D图表）库

  1. 为什么使用？
    • 数据可视化
    • js库，D3，echarts
  2. 三层结构
    • 容器层：1.画板层（Canvas） 2.画布层（Figure）：plt.figure() 3.绘图区/坐标系（axes）：figure,axes=plt.subplots(nrows=,ncols=,figsize=,dpi=)
    • 辅助显示层：网格、刻度、图例、标题等辅助显示，使图表清晰易懂
      • 添加网格plt.grid()
      • 显示图例 plt.legend()
      • 修改坐标轴刻度信息 plt.xticks() plt.yticks(y,**args)
      • 添加描述信息 plt.x/ylabel(),plt.title()
    • 图像层：画各种类型、风格图表
      • 折线图 plt.plot()
      • 散点图 plt.scatter()
      • 柱状图 plt.bar()
      • 直方图 plt.hist()
      • 饼图 plt.pie()
  3. 折线图绘制
    • 导包，包含了一系列绘图函数import matplotlib.pyplot as plt
    • 图表保存 plt.savefig(“test1.png”)
    • 绘制画布 plt.figure()
    • 绘制折线 plt.plot(x,y)
    • 设置坐标轴标签，图表标题 plt.xlabel() plt.ylabel() plt.title()
    • 图表显示 plt.show()
import matplotlib.pyplot as plt  #导包，包含了一系列绘图函数
%matplotlib inline  #环境，有些不需要
import random
x = range(60)
y_shanghai = [random.uniform(15,18) for i in x]
y_nanchang = [random.uniform(10,16) for i in x]
plt.figure(figsize=(20,8),dpi=80) #设置大小和清晰度
plt.plot(x,y_shanghai,color='r',linestyle='-.',label='shanghai')
plt.plot(x,y_nanchang,color='b',linestyle='--',label='nanchang')
#显示图例
plt.legend(loc='lower left') 
#修改坐标刻度值
x_lable =["11点{}分".format(i) for i in x] #x轴说明
plt.xlabel("time") #x轴标签
plt.ylabel("temp") #y轴标签
plt.title("picture") #图表标题
plt.yticks(range(0,40,5))#y轴0-40，隔5
plt.xticks(x[::5],x_lable)
#添加网格显示
plt.grid(True,linestyle='--',alpha=0.5)#风格和透明度，true可省略
plt.show()#之后会释放所有图像资源

  1. 多个坐标系显示
    • Plt.subplots(面向对象的画图方法)
    • figure,axes=Matplotlib.pyplot.subplots(nrows=1,ncols=1,**fig_kw)
    • axes[0].方法名（）
import matplotlib.pyplot as plt  #导包，包含了一系列绘图函数
import random
x = range(60)
y_shanghai = [random.uniform(15,18) for i in x]
y_nanchang = [random.uniform(10,16) for i in x]
figure,axes = plt.subplots(nrows=1,ncols=2,figsize=(20,8),dpi=80)
axes[0].plot(x,y_shanghai,color='r',linestyle='-.',label='shanghai')
axes[1].plot(x,y_nanchang,color='b',linestyle='--',label='nanchang')

axes[0].legend(loc='lower left')#显示图例
axes[1].legend(loc='lower left')#显示图例

#描述信息
axes[0].set_xlabel("time") #x轴标签
axes[0].set_ylabel("temp") #y轴标签
axes[1].set_xlabel("time") #x轴标签
axes[1].set_ylabel("temp") #y轴标签
axes[1].set_title("picture nanchang") #图表标题
axes[0].set_title("pictureshanghai") #图表标题

#修改刻度值
x_lable =["11点{}分".format(i) for i in x] #x轴说明
axes[0].set_yticks(range(0,40,5))
axes[0].set_xticks(x[::5])
axes[0].set_xticklabels(x_lable)
axes[1].set_yticks(range(0,40,5))
axes[1].set_xticks(x[::5])
axes[1].set_xticklabels(x_lable)

#添加网格显示
axes[0].grid(True,linestyle='--',alpha=0.5)#风格和透明度，true可省略
axes[1].grid(True,linestyle='--',alpha=0.5)#风格和透明度，true可省略
plt.show()

  1. 直方图 (histogram)
    • 首先对数据进行分组，然后统计每个分组内数据元的数量，横坐标标出每个组的端点，纵坐标表示频数，每个矩形的高代表对应的频数
    • 直方图有助于了解数据的分布情况，诸如众数、中位数的大致情况、数据是否存在缺口或者异常值。
    • 组数=极差/组距=（max-min）/bins
    • plt.hist(x,bins=None（组数）,normed=none(是否显示频率,最新density=True)，**kwargs)
time = [131,120,120,150,90,117,108,135,102,132,156,112,139,98,111,156,120,130,141,97,94,113]
plt.figure(figsize=(20,8),dpi=80)
distance=5
group_num=int((max(time)-min(time))/distance)
plt.hist(time,bins=group_num,density=True)
plt.xticks(range(min(time),max(time)+5,distance))
plt.show()
  1. 饼图（pie）
    • Plt.pie(x,labels=(每部分名称),autopct=（占比显示指定%1.2f%%）,colors)
    • plt.axis(‘equal’)：为了让显示饼图保持圆形
movie_name = ['雷神','aa','bb','cc','dd']
tickets = [123,232,555,100,200]
plt.figure(figsize=(20,8),dpi=80)
plt.pie(tickets,labels=movie_name,colors=['b','r','y','k','c'],autopct='%1.2f%%')
plt.axis('equal')
plt.legend()
plt.show()
