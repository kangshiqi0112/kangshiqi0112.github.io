基础知识

与c语言的小区别

  • 1.c语言没有提供bool类型，c++有bool类型（true,false=0），以此类型输出需加boolalpha
  • 2.两种初始化方法 int x=2 和int x(2),c++可以随用随定义
  • 3.c语言用scanf输入，print输出；c++用cin输入，cout输出。（cout<<x<<endl;）,endl(类似于\n回车)；（cin>>x>>y）
  • 4.八进制输出（cout<<oct<<x）,十进制（dec）,十六进制（hex）
  • 5.面向对象三大特征：封装、多态、继承

其他细节

  • 1.namespace(命名空间)，用来区分来自不同地方的相同名字的变量和函数等
  • 2.#include 头文件 内包含输入输出语法，但是它在空间std中，需要using namespace std;
include<iostream>//包含输入输出语法
include<stdlib.h>
using namespace std;
namespace A
{
    int x=0;
    void f1(int *arr){int a=arr[0]}//数组传参
    void f2();
}
int main(){
cout<<A::x<<endl;
A::f1();
system("pause");//请按任意键继续。。。
}
  • 3.引用类型（变量的别名）
    • 引用必须初始化（int a =2;int &b=a;），对别名的修改会影响到本身
    • 指针类型的引用（*&指针引用名=指针;）int a=10; int *p=&a; int *&q=p; *q=20; cout <<a;—-结果为20
    • 函数参数的引用，比指针更方便（void get(int &x,int &y){}）
  • 4.const
    • const int x=3//常量，无法更改
    • int const p(const int *p)=&x,此时p可以指向另一个地址，const修饰的是p,即x的值
    • int *const p=&x,此时p的指向不能变，x的值可变
    • const int &y=x,此时通过变量名x修改可以，引用名Y不行
    • #define 是宏定义，编译时不检查语法错误，const检查
  • 5.c++中函数参数可以有默认值，但含有默认值的参数要放到后面，使用时最好函数声明时写上默认值，定义时不写，实参会覆盖默认值
  • 6.函数重载：在同一作用域中，函数名相同，函数参数不同
  • 7.内联函数inline：编译时将函数体代码和实参代替函数调用语句(声明：inline int max();)。递归函数无法使用内联函数，内联编译是建议性的，由编译器决定。逻辑简单、调用频繁的函数建议使用内联函数
  • 8.内存管理：内存的申请与释放（int *p=new int;int *arr=new int[10]//申请块内存）（delete p/[]p;p=null）（c中void *malloc()–void free()）
  • 9.String 类型 头文件(#include)
    • 初始化 string name=”dd”;string hobby(“football”);string s2(s1);string s4(n,’c’)//n个c;
    • s.empty()//判断S是否为空串
    • s.size()//返回s中字符的个数
    • 字符串连接中””+s1+””,s1+s2合法，但是””+””不合法
  • 10.内存分区：栈区（变量，指针，由系统控制），堆区（new），全局区（存储全局变量及静态变量），常量区（string），代码区（存储逻辑代码的二进制）

封装（类与对象）

  • 1.类中访问限定符：public（公共），protected（受保护的），private（私有的）
  • 2.实例化对象：
    • 从栈中实例化
    • 从堆中实例化

  • 3.通常将方法设为公有属性，将数据成员设为私有属性，以便封装数据成员（定义set,get方法除了可以保护数据，还可以对传入的数据做判断）
  • 4.类内定义，在类里定义的函数，编译器会优先将它们作为内联函数（inline）
  • 5.类外定义（函数体写在类外）
    • 同文件类外定义（函数与类（car）在同一文件，void car::run(){}）
    • 分文件类外定义（推荐使用，car.h文件中写类，car.cpp文件中#include”car.h”和方法void car::run(){}）
  • 6.对象初始化，有的数据只需要初始化一次，此时使用构造函数，对象会自动调用
  • 7.构造函数没有返回值，可以重载，且仅会被调用一次。当用户没有定义时，编译器会自动生成一个构造函数，构造函数名称与类名一致
  • 8.初始化列表（student():m_strname(“jim”),m_intAge(10){}）,只能用于构造函数，先于构造函数执行，可以给类中定义的常量（const）初始化赋值
  • 9.拷贝构造函数（student(const student&stu){}）系统会自动生成（拷贝和普通系统只会生成一个）,当采用直接初始化或复制初始化实例对象时，系统自动调用拷贝构造函数
  • 10.析构函数（~类名(){}），对象销毁前自动调用，释放资源。没有自定义的话系统会自动生成，且不允许有参数
  • 11.浅拷贝，当出现指针时，会产生麻烦
  • 12.深拷贝，赋值时新建另一个堆，将每一个值都复制过去
  • 13.this指针，取当前对象的地址（*this是对象）
  • 14.常对象成员（初始化列表赋值）和常成员函数（不可修改成员的值，重载时只有常对象才能调用常成员函数，也只能调用常成员函数。普通对象都可调用）

    继承

  • 1.公有继承（class A:public B{}）,B可以调用A中的属性和方法,继承时该是什么类型就是什么类型
  • 2.保护继承（class A:protected B{}），A中protected和public的成员被继承到B中后是protected类型
  • 3.私有继承（class A:private B{}），A中protected和public成员被继承到B中后是private类型
  • 4.无论哪种继承方式，A中私有属性的成员被继承到B中后处于不可见位置，B无法访问
  • 5.隐藏 子类与父类有同名函数时，从父类继承过来的函数（或数据成员）将被隐藏，但是可以通过特殊方式使用（对象.父类::函数（））
  • 6.虚析构函数（virtual~），父类使用虚析构函数，此时将子类赋值给父类定义的指针的情况下释放指针时不会遗漏子类的析构函数，否则会造成内存泄漏等问题
  • 7.多重继承（子孙关系），一个子类同时有多个父类称为多继承
  • 8.虚继承（：virtual public A()），多继承+多重继承时（菱形继承），子类继承父类时使用虚继承，可以使孙类减少冗余和出现重定义的问题

    多态（虚函数）

  • 1.多态：相同对象收到不同消息或不同对象收到相同消息时产生不同的动作
  • 2.静态多态（早绑定），通常为函数重载
  • 3.动态多态，以封装和继承为基础，父类指针指向子类对象，通过父类指针完成动作（父类的成员函数为虚函数才行）
  • 4.普通函数、静态成员函数（static）、内联函数、构造函数不能是虚函数
  • 5.函数覆盖 虚函数继承后，子类的虚函数表中的内容会覆盖父类中的同名内容，实现多态
  • 6.纯虚函数（virtual double A()=0;）,含有纯虚函数的类为抽象类，抽象类无法实例化对象
  • 7.接口类：仅含有纯虚函数的类
  • 8.RTTI（运行时类型识别）（typeid(obj).name();typeid(*obj)==typeid(类名);Bird *bird=dynamic_cast<Bird>(obj);注：只能用于指针或引用的转换）
  • 9.举例 try{f1()}…catch（int/…）{} f1(){throw 1 }抛出异常
  • 10.常见的异常：数组下标越界、除数为0、内存不足

    模板

  • 1.友元函数(friend),容易破坏封装性
  • 2.友元关系不能传递，友元关系的单向性
  • 3.静态数据成员/成员函数（static），实例化之前就存在，可以通过类使用，静态成员函数无法调用非静态数据成员，因为没有传默认的this指针
  • 4.运算符重载（类中 对-重载：类名& operator-();-对象名();）,-(取反),++,–
  • 5.二元运算符重载，A& operator+(A &a1,A &a2);friend ostream& operator<<(ostream &out,const A &a1);int operator[](int index);
  • 6.函数模板 template<typename T,int n>void dispaly(T a){…cout<<n} dispaly(10);
  • 7.类模板 templateclass A{public:dispaly();}private:T *m_pArr;templatevoid A::display(){};Aarr;
  • 8.模板代码不能分离编译，即不能分成.h文件
  • 9.c++标准模板库（STL）
    • vector 向量（对数组的封装）：vector v1;empty();begin();end();clear();front();back();size()…（迭代器interator）
    • list 链表（数据插入速度快）
    • map 映射（key–value）
